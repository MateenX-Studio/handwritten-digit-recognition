# Handwritten Digit Recognition System

A deep learning web application that recognizes handwritten digits (0-9) using CNN trained on MNIST dataset.

## Accuracy: 98.93%

## Features
- Draw digit on canvas
- Upload image for prediction
- Camera capture support
- Multi-digit detection

## Tech Stack
- Python, TensorFlow, Keras
- OpenCV, Streamlit
- MNIST Dataset (70,000 images)

## How to Run
pip install -r requirements.txt
streamlit run app.py
